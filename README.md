# Internship Management — Event-Driven Microservice System

A microservice-based system for managing student internships, built using Domain-Driven Design (DDD), CQRS, and Event Sourcing principles.

## Tech Stack
Kotlin, Spring Boot, Axon Framework, Kafka, PostgreSQL, Feign Client, Consul, Keycloak

## Architecture
The system follows an event-driven microservice architecture with asynchronous communication and strong separation of concerns using CQRS and Event Sourcing.

## Key Features
- Microservice-based internship management system  
- Event-driven communication using Apache Kafka  
- Service discovery with Consul  
- Secure authentication and authorization with Keycloak  
- Inter-service communication via Feign Client  
- CQRS and Event Sourcing with Axon Framework  

## Domain Model
I was responsible for implementing the **JournalEntry** aggregate and its associated business logic within the domain model.
